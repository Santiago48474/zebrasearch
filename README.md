# ZebraSearch (версия на DuckDuckGo)

Свой поисковик со своим названием и оформлением. Работает через DuckDuckGo — без API-ключей, без биллинга, без привязки карты.

## Установка в Termux

```bash
pkg update && pkg upgrade -y
pkg install nodejs -y

cd ~/zebrasearch
npm install
node server.js
```

Открой в браузере на телефоне:
```
http://localhost:3000
```

Готово — больше ничего настраивать не нужно.

## Структура проекта

```
zebrasearch/
├── server.js          # бэкенд — обращается к DuckDuckGo и отдаёт результаты
├── package.json
├── .env.example        # только порт, ключи не нужны
└── public/
    ├── index.html      # главная страница
    ├── style.css       # чёрный размытый фон + пузырчатые кнопки
    ├── app.js           # логика поиска
    └── logo.svg         # аватарка (чёрный круг, белые полосы)
```

## Как это работает

- Браузер обращается только к твоему серверу (`/api/search`), а не к DuckDuckGo напрямую.
- Сервер сам стучится в html-версию DuckDuckGo и вытаскивает из неё заголовки, ссылки и описания.
- Никаких ключей, аккаунтов или платёжных данных не требуется.
- Внизу поисковой строки — мелкая подпись "duckduckgo", как отсылка к источнику данных.

## Возможные ограничения

- DuckDuckGo может иногда менять разметку своей html-страницы — если результаты вдруг перестанут находиться, разметку в `server.js` (функция `parseResults`) нужно будет немного поправить.
- Слишком частые запросы с одного IP могут временно блокироваться DuckDuckGo — для личного использования это маловероятно.

## Что можно доработать дальше

- Голосовой поиск (Web Speech API)
